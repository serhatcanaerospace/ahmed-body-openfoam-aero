# Ahmed Body OpenFOAM Aero

![OpenFOAM](https://img.shields.io/badge/OpenFOAM-CFD-1F6FEB?style=flat) ![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=flat) ![Status](https://img.shields.io/badge/status-active-brightgreen?style=flat)

Public roadmap repository for an Ahmed body external aerodynamics benchmark in
OpenFOAM. The project will focus on drag, wake structure and mesh sensitivity.

## Literature basis

See [docs/literature.md](docs/literature.md).

## Planned deliverables

- Parametric Ahmed body geometry.
- OpenFOAM case generator.
- Drag and pressure coefficient extraction.
- Wake slice and streamline visualizations.
- Mesh convergence report.
